# Quickstart for "Create AI Textbook"

This guide provides instructions on how to set up and run the Docusaurus project for the AI Textbook.

## Prerequisites

-   Node.js (LTS version)
-   npm or yarn

## Installation

1.  Navigate to the `ai-book` directory:
    ```bash
    cd ai-book
    ```
2.  Install the dependencies:
    ```bash
    npm install
    ```
    or
    ```bash
    yarn install
    ```

## Running the Development Server

1.  From the `ai-book` directory, run the following command:
    ```bash
    npm start
    ```
    or
    ```bash
    yarn start
    ```
2.  Open your browser and navigate to `http://localhost:3000`.

## Building the Project

1.  From the `ai-book` directory, run the following command:
    ```bash
    npm run build
    ```
    or
    ```bash
    yarn build
    ```
The static files will be generated in the `ai-book/build` directory.
